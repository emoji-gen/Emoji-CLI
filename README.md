## Emoji-CLI

:tada: Ultimate Emoji Generator for CLI

## Requirements

- Python 3.5
- pip

## Getting Started

```
$ pip install git+https://github.com/emoji-gen/Emoji-CLI.git
```

## Usage

```
$ emoji-gen 天才 現る
```
