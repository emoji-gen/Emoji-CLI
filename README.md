## Emoji Generator CLI

:tada: The ultimate command line client for Emoji Generator

## Requirements

- CPython 3.5, 3.6, or 3.7
- pip

## Getting Started

```
$ pip install git+https://github.com/emoji-gen/cli.git
```

## Usage

```
$ emoji-gen 天才 現る
```
